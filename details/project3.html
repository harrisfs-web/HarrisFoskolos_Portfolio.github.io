Tennis Data Scraping, Analysis and Visualization

This project focuses on analysis of tennis results to create detailed head-to-head analysis of the players. The purpose is to demonstrate separately the functionality of gathering data through web scraping, transforming the data using queries and Excel formulas, and creating a dashboard for analysis of the results. The queries have been decoupled from the Excel file since the data source is not mentioned in this repository.

Tools used: Excel, Power Query (Web Scraping)
