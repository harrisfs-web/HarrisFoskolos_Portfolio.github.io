News Analysis in Microsoft Fabric

The goal was to create an end-to-end data engineering project within the Fabric environment that loads news data and analyzes their sentiment. The steps include data loading data from JSON responses into delta tables, analyzing them (Synapse ML) and creating a report.

Tools used: MS Fabric (Dataflows, Lakehouse, PySpark), Synapse ML (sentiment analysis model), PowerBI, Figma
